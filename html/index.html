Setup Guide

  1. Enter camera credentials and Save. A viewer-level account is enough — the app reads snapshots/RTSP via 127.0.0.1.
  2. App publishes the HomeKit accessory. It restarts and announces itself via mDNS (look for "published" in the CamScripter log).
  3. Pair in the Home app (iPhone on the same LAN): + → Add Accessory → More options… → select the camera → ignore the "uncertified" warning → enter PIN 0314-5154.
  4. Done. The tile shows live snapshots; tap for live stream (if enabled). To re-pair, delete the hap/ folder in package data and restart.

HomeKit

VMD and AOA each appear as their own HomeKit Motion Sensor ("VMD Motion Sensor" / "AOA Motion Sensor"), so you get separate notifications and automation triggers per detector. The detection apps must be running with at least one profile/scenario configured on the camera. Hold time keeps a sensor "detected" for a few seconds after motion stops, so a flapping detector doesn't spam Home notifications (0 = report raw on/off).

To get push alerts: in the Home app open this Motion Sensor → Settings (gear) → Status and Notifications → enable Allow Notifications.
